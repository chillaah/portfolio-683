astar-visual

This project is a C++ implementation of the A* pathfinding algorithm, visualized using the SFML library. It aims to demonstrate the algorithm's process of finding the shortest path between a starting and goal cell on a 20x20 grid, while avoiding obstacles, through an interactive step-by-step visualization.

pioneer-mobile-robot-slam

This project aims to implement Simultaneous Localization and Mapping (SLAM) on a Pioneer 3-DX mobile robot. The project comprises four modules: robot motion and control, planning, Bayesian filtering and localization, and EKF SLAM implementation using real data.
